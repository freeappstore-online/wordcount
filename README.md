# Word Counter

A fast, privacy-friendly word counter and text analyzer. Live stats, frequency analysis, find & replace, transforms, line sorting, cleanup tools, saved snippets — all client-side, nothing leaves your browser.

- **Live**: https://wordcount.freeappstore.online
- **Stack**: React 19 + Vite 6 + Tailwind 4 + TypeScript (strict)
- **License**: MIT

## Develop

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Part of [FreeAppStore](https://freeappstore.online) — free forever.
